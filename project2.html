Задача

Разработать сверхточный счетчик, позволяющий измерять потребление газа с помощью современных ультразвуковых технологий, а также передавать показания по радиоканалу

разработка

чем наделили устройство?

электроакустические преобразователи

Счетчик газа выполнен из соединенных через герметизирующую прокладку основания и верхней крышки. Внутри счетчика газа образован измерительный канал. В торцах измерительного канала закреплены два электроакустических преобразователя (печатные платы с установленными на них ультразвуковыми датчиками) с герметизированными выводами

Модуль измерений и индикации

В состав устройства добавили модуль измерений и индикации с ЖКИ, автономным источником питания и датчиком температуры

Модуль передачи данных

Добавили модуль передачи данных (радиомодуль от 863,0 до 870,0 МГц) с автономным источником питания

Режимы отображения информации

Под лицевой панелью счётчика на плате измерений и индикации расположили сенсорную кнопку, предназначенную для изменения режимов отображения информации на ЖКИ дисплее

Что было достигнуто?

  • Получили евразийский патент на изобретение «Устройство и способ определения фазы сигнала ультразвукового датчика расходомера», примененный в ультразвуковом счётчике газа
  • При разработке функциональной схемы подбирались компоненты и протоколы, принято решение производить передачу данных с использованием протокола NB-Fi
  • Измерение счётчиком объёма газа, приведенного к стандартным условиям, выполнили косвенным методом динамических измерений. Метод основан на преобразовании объёмного расхода газа преобразователем расхода ультразвуковым в сигнал измерительной информации с автоматическим вычислением объёма газа при рабочих условиях и приведении их к стандартным условиям с помощью электронного устройства, используя температуру, полученную от датчика температуры
  • Реализовали в счётчике ультразвуковой времяимпульсный метод по способу измерения объёмного расхода газа в рабочих условиях

разработка программного обеспечения

Особености Разработки ПО

Разработали программное обеспечение, встраиваемое в микроконтроллер счётчика для отсчёта, интегрирования (накопления) и перевода в объём газа величины разности времени перемещения коротких импульсов по направлению потока газа, с последующим отображением измеренного значения на устройстве индикации и передачи его по интерфейсу связи.

Разработали алгоритм работы пользовательского интерфейса, в соответствии с которым ПО счётчика осуществляет отображение запрашиваемой информации из энергонезависимой памяти на ЖКИ при физическом воздействии на элемент управления (сенсорную кнопку).

Результат

Разработан ультразвуковой счётчик газа, возможности которого превышают исходные требования клиента, получен патент на изобретение.

По итогам работ счётчик предназначен для измерения, сохранения в запоминающем устройстве, отображения на дисплее информации об объёме потреблённого газа, проходящего через счётчик, и передачи этой информации по радиоканалу на базовую станцию.

Устройство производится серийно.

Особености Разработки ПО

Разработали программное обеспечение, встраиваемое в микроконтроллер счётчика для отсчета числа оборотов крыльчатки, количество оборотов которой пропорционально объему воды, с последующим отображением данных на ЖКИ, накоплением и сохранением собранных данных, а также передачи данных по радиоканалу. Для разработки программы (прошивки) для контроллера использовалась среда разработки (IDE) IAR.

Разработали алгоритм работы пользовательского интерфейса, предназначенный для изменения режимов отображения информации на ЖКИ дисплее счётчика при касании к сенсорной кнопке.

Разработали собственную программную платформу, предоставляющую необходимый набор функций, реализующих возможность модульной разработки программного обеспечения счётчика.

Построили профили энергопотребления для различных сценариев работы устройства, оптимизирующие ПО с точки зрения энергосбережения.
